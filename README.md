# App Store Reviews Downloader

Небольшой скрипт на Python, который скачивает отзывы к игре/приложению из App Store и сохраняет их в текстовый файл.

## Что делает скрипт

Скрипт:

- спрашивает:
  1. URL игры/приложения в App Store  
  2. код страны стора (например: `us`, `ru`, `de`, `fr`)  
  3. оценку отзывов (от `1` до `5`) или пусто — тогда берёт все
- загружает отзывы через публичное RSS-API App Store
- сохраняет отфильтрованные отзывы в файл вида:

`reviews_<app_id>_<country>.txt`  
например: `reviews_1195601592_us.txt`

## Установка

Требуется установленный **Python 3**.

1. Перейти в папку проекта в терминале:
    cd /Users/<твоё_имя>/Documents/python/ratings
2. Создать виртуальное окружение (один раз):
    python3 -m venv venv
3. Активировать окружение:
    source venv/bin/activate
4. Установить зависимости:
    pip install -r requirements.txt

## Запуск

В активированном окружении:
    python ratings.py

Скрипт последовательно спросит:

1. URL игры из App Store
2. код страны (us, ru, de и т.д.)
3. оценку отзывов (1–5) или пусто — для всех оценок

После работы скрипта в папке проекта появится файл с отзывами, например:
reviews_1195601592_us.txt

## Выход из окружения

Чтобы выйти из виртуального окружения:
    deactivate
